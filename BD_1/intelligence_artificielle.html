Étape 1

Onshape to Robot

Dans un premier temps, il faut convertir l’assemblage 3D de BD-1 vers un fichier URDF ( Unified Robot Description Format ) pour permettre qu’il soit interprété correctement par l’application Isaac Sim. Comme mon modèle 3D se trouvait sur Onshape, j’ai utilisé Onshape to Robot. Il faut faire attention à bien nommer chacun des joints pour assurer une bonne conversion via Onshape to Robot.
