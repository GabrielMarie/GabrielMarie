ℹ️ Lecture rapide

  • Kp 0–2 : visible surtout > 60–65°N
  • Kp 3–4 : possible > 55–60°N
  • Kp 5 : parfois 50–55°N
  • Kp 6–7 : 45–50°N
  • Kp ≥8 : grandes tempêtes

OVATION tient compte de l’ovale auroral et donne une proba locale en %. La précision dépend des conditions.

Dépannage (ouvrir)
  • Si rien ne s’affiche, vérifie que ton navigateur charge bien les JSON NOAA (CORS). Le code essaie d’abord direct, puis via un proxy Cloudflare si configuré.
  • Définis PROXY_BASE dans le code si tu as un worker. Exemple : https://ton-worker.workers.dev/?url=
  • Ouvre la console (F12) pour voir les erreurs et les URLs réellement appelées.
